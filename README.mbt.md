# moonbitlang/maria

**maria** is an initialism for **m**oon **a**gent **r**ewritten **i**n **a**sync, which utilizes [moonbitlang/async](https://github.com/moonbitlang/async).

## Development Status

This project is still in early development. The API and functionality may
change significantly in future releases.

As for now, we only support macOS (arm64) and Linux (x86_64). Windows support is
coming soon.

## Usage

```bash
$ export OPENAI_API_KEY=<your_api_key>
$ moon update
$ moon run . -- Hello
Hello! How are you doing today? Is there anything I can help you with?
```
